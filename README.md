# 2uv

  [UV]: https://docs.astral.sh/uv/

Migrate Python projects from other package managers to [UV].

    $ uvx 2uv .

__Features__

* [ ] Convert `pyproject.toml` to UV + Hatchling from
    * [ ] Poetry
    * [ ] PDM
    * [ ] Pipenv
* [ ] Convert lock files to `uv.lock` from
    * [ ] `poetry.lock`
    * [ ] `pdm.lock`
    * [ ] `Pipfile.lock`
